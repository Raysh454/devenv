# Devenv

## Require Packages
- Hyperland
- Waybar
- pywal
- wofi
- cliphist
- wl-clipboard
- swww
- nm-applet
- dunst
- flameshot
- scrot

## Installation

For Arch Linux, simply:
```bash
chmod +x install.sh
./install.sh
```

For other distros, install the required packages then:
```bash
chmod +x install.sh
./install.sh copy
```

For nvim only:
```bash
chmod +x install.sh
./install.sh nvim
```
